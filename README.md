# Bluetooth-Controlled-Intelligent-Roller-Shutter
Bluetooth-Controlled Intelligent Roller Shutter using Arduino. The objective of the project was to automate the operation of roller shutters using a smartphone instead of manual control.

The system consists of an Arduino Uno, an HC-05 Bluetooth module, a motor driver, and a DC motor connected to the roller shutter mechanism. The user sends commands from a smartphone application via Bluetooth. The HC-05 module receives these commands and forwards them to the Arduino through serial communication.

Based on the received command, the Arduino controls the motor driver, which rotates the motor either clockwise or counterclockwise to open or close the shutter.

Through this project, I gained practical experience in Arduino programming, Bluetooth communication, embedded systems, and motor control. It also helped me understand how hardware and software can be integrated to create automation solutions
