# Bluetooth-Controlled Intelligent Roller Shutter

## Overview

The Bluetooth-Controlled Intelligent Roller Shutter is an Arduino-based automation system that enables users to open and close roller shutters using a smartphone. The project replaces traditional manual operation with a wireless control mechanism, improving convenience and accessibility.

## Objective

The objective of this project is to automate roller shutter operation through Bluetooth communication, allowing users to control the shutter remotely using a mobile device.

## System Architecture

The system consists of an Arduino Uno, HC-05 Bluetooth module, L298N motor driver, and a DC motor connected to the roller shutter mechanism.

1. The user sends commands from a smartphone application.
2. The HC-05 Bluetooth module receives the commands and forwards them to the Arduino Uno.
3. The Arduino processes the commands and controls the motor driver.
4. The motor rotates clockwise or counterclockwise to open or close the roller shutter.
